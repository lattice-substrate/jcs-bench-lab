\caption{Per-iteration latency for \texttt{array-2048} (Rust API
canonicalize, $n{=}20$ measured iterations after 3~discarded warmup
iterations).  \texttt{array-2048} is the largest number-dense workload
in the standard set (98{,}901~bytes, $\nu{=}40\%$); if warmup stabilizes
here, smaller workloads will stabilize at least as fast.
\schubfach-rs shows residual cache stabilization in iterations 1--4;
\burgerdybvig-rs is stable from iteration~1.  Both reach steady state by
iteration~5, confirming that 3~warmup iterations are sufficient and any
residual warmup effect in the first few measured iterations does not bias
the within-language comparison (both implementations share the same
harness).}
